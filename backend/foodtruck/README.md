## 주요 기능

### 사용자

- 행사에 등록된 푸드트럭 목록 및 메뉴 조회
- 메뉴 주문/취소
- 주문 처리 상태 조회

### 사장님

- 회원가입/로그인
- 처리 상태별 주문 목록 조회
- 주문 처리 상태 변경
- 메뉴 관리(품절 처리)
- 새 주문 실시간 push 알림
